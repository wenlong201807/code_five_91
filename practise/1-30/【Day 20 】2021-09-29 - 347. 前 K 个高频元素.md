# 题目名称

- [347. 前 K 个高频元素](https://leetcode-cn.com/problems/top-k-frequent-elements/)

## 解题思路

```javascript

map特点运用

```

### 代码块

```javascript

var topKFrequent = function (nums, k) {
  let map = new Map()
  let arr = [...new Set(nums)]

  nums.map((num) => {
    if (map.has(num))
      map.set(num, map.get(num) + 1)
    else
      map.set(num, 1)
  })

  return arr.sort((a, b) => map.get(b) - map.get(a)).slice(0, k)
};

```

### 时间复杂度和空间复杂度

- 时间复杂度 O(n)
- 空间复杂度 O(1)
